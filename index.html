02 — About

Small studio. Wide range.

VXVO is an independent studio. The remit is broad on purpose — productivity tools one week, an arcade game the next, a translator the week after. The throughline is taste, polish, and shipping.

Everything is designed and built end-to-end: visual identity, frontend, backend, mobile, and the boring infrastructure that keeps it running. The studio works in tight loops with users and ships small.

  • Web Next.js, React, TypeScript, edge runtimes
  • Mobile Native + cross-platform on iOS and Android
  • AI Claude, GPT, structured generation, agents
  • Backend Supabase, Postgres, Cloudflare Workers
  • Design Identity, product UI, motion, illustration
  • Games Original titles, sandbox systems, arcade
